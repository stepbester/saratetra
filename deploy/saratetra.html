SARATETRA
Description

This simple tetris clone is written in JavaScript and uses the HTML5 canvas for graphics.

Scoring

Earn points by clearing lines!

1 line 10 pts
2 lines 25 pts
3 lines 75 pts
4 lines 300 pts

Points are multiplied by the number of the level.

Controls
'X' key
Start the game.
'P' key
Pause/unpause the game.
Left/right arrow
Move the tetromino left or right
Up arrow
Rotate the tetromino
Down arrow
Force-drop the tetromino
